# 技能图标资源

## 目录用途
存放所有主动技能、被动技能、光环的图标。

## 图片规格要求
- **格式**: PNG（支持透明通道）
- **尺寸**: 64×64（推荐）或 128×128（高清变体）
- **风格**: 暗黑风格技能图标 — 鲜明的图形剪影 + 发光效果
- **配色**: 根据技能元素/类型着色（红色=火，蓝=冰，黄=电，绿=毒，紫=奥术，金色=神圣，暗紫=暗影）
- **背景**: 透明或暗色圆形底衬

## 引擎内使用
- 在 Godot 中创建 `TextureRect` 节点，将 PNG 拖入 `texture` 属性
- 技能按钮使用 `TextureButton`，设置 `texture_normal`、`texture_pressed`、`texture_disabled`
- 图标尺寸 64×64 适配 1280×720 分辨率的技能栏
