# 属性图标资源

## 目录用途
存放所有角色属性、抗性、伤害类型的图标，用于角色面板和物品词缀显示。

## 图片规格要求
- **格式**: PNG（支持透明通道）
- **尺寸**: 32×32（小图标）或 48×48（中图标）
- **风格**: 扁平图标风格，单色剪影 + 柔和发光
- **配色**: 每个属性有固定配色方案（见下表）

## 核心属性（可分配）

| 资源名称 | 描述 | 配色 | 优先级 |
|----------|------|------|--------|
| `strength.png` | 力量 — 拳头/肌肉臂 | 红色 #cc4444 | P0 |
| `dexterity.png` | 敏捷 — 飞矢/羽毛 | 绿色 #44cc44 | P0 |
| `vitality.png` | 体力 — 心脏/盾牌 | 棕色 #cc8844 | P0 |
| `energy.png` | 能量 — 法力球/星星 | 蓝色 #4444cc | P0 |

## 引擎内使用
- 在角色面板中使用 `TextureRect` 显示属性图标
- 物品词缀旁用对应抗性/伤害图标增强可读性
- 可在 Godot 中创建 `AtlasTexture` 将所有属性图标整合为图集
