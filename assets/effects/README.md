# 特效资源

## 目录用途
存放所有视觉特效：战斗打击效果、掉落光束、升级特效、转生特效等。

## 图片规格要求
- **格式**: PNG（支持透明通道）
- **风格**: 暗黑风格粒子/光效 — 高饱和强调色 + 透明渐变边缘
- **形式**: 精灵表（Sprite Sheet）或单帧均可；推荐精灵表用于序列帧动画
- **混合模式**: 在 Godot 中使用 `Add` 混合模式（适合发光效果）

## 引擎内使用
- 序列帧动画：使用 `AnimatedSprite2D`，配置 `sprite_frames` 资源，设置 `speed_scale`
- 单帧特效：使用 `Sprite2D` + `Tween`（缩放/淡出动画）
- 粒子效果：创建 `GPUParticles2D` 节点，配置 `process_material`（ParticleProcessMaterial），拖入纹理
- 发光混合：在 CanvasItem 的 `Material` 中设置 `blend_mode = BLEND_MODE_ADD`
