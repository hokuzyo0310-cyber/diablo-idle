# 角色贴图资源

## 目录用途
存放游戏中所有角色、怪物、BOSS 的精灵/贴图资源。

## 图片规格要求
- **格式**: PNG（支持透明通道）
- **风格**: 暗黑哥特风格，暗色调为主 [待确认：像素风 / 手绘插图风 / 极简几何风？]
- **姿势**: 闲置待机姿态（挂机游戏角色为静态显示，不需要行走/攻击动画帧）
- **光源**: 顶光或前侧光，暗调为主，高光用于强调轮廓

## 风格参考
- **暗黑破坏神 2** 哥特风格：暗色、粗犷、恐怖氛围
- **色板参考**：深红 #8b0000、暗金 #8b7355、骨白 #d4c5a9、深渊黑 #0a0808
- **轮廓要清晰**，便于在小尺寸下识别

## 引擎内使用
- 贴图存入此目录后，去除 `assets/` 目录的 `.gdignore` 文件（或移动到子目录 `.gdignore`）
- 在 Godot 中创建 `Sprite2D` 节点，将 PNG 拖入 `texture` 属性
- 战斗中角色为静态显示，不需要 `AnimatedSprite2D`（可后续扩展）
