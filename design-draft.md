## 1. 游戏概述

### 一句话描述
暗黑破坏神主题的挂机刷宝游戏 — 玩家选择英雄职业，自动战斗击杀怪物，收集随机词缀装备，挑战更深层次的地下城，通过转生获得永久力量。

### 定位
- **类型**: 放置/挂机 + ARPG 掉落深度
- **差异化**: 将暗黑破坏神式的深度物品化系统（前后缀词缀、独特能力、远古物品）放入可挂机的轻量框架中
- **目标体验**: 打开游戏 → 装备刷刷刷 → 满足收集欲 → 关闭等离线收益 → 回来变得更强

1. **游戏节奏**: 
   - [ ] A. 纯挂机 — 游戏打开后完全自动，玩家只需偶尔更换装备
   - [ ] B. 半主动 — 挂机为主，但需要手动分配属性点、选择技能、鉴定装备
   - [ ] C. 混合模式 — 平时挂机，Boss 战/特定阶段需要手动操作

2. **单次游戏时长**:
   - [ ] A. 5-10 分钟快速 check-in（适合移动端碎片时间）
   - [ ] B. 30 分钟+ 中度游戏（适合 PC 端专注游玩）

   - [ ] B. 挂机游戏玩家（追求轻松放置和数字增长）
   - [ ] C. 两者兼顾

## 2. 核心游戏循环

### 主循环
```
┌─────────────────────────────────────────────────────────┐
│                    挂机战斗（自动）                        │
│  角色自动攻击当前敌人 → 击杀 → 掉落战利品 → 下一只         │
│                    ↓                                    │
│              收集 & 鉴定装备                              │
│    （查看词缀 → 对比属性 → 装备/出售/分解）                │
│                    ↓                                    │
│                角色变强                                   │
│   （升级属性 + 装备提升 → DPS/金币产出增加）                │
│                    ↓                                    │
│                 推进关卡                                  │
│    （更高区域等级 → 解锁新词缀层级 → 更好的掉落）            │
│                    ↓                                    │
│               击败 BOSS                                   │
│     （每 5 阶段 = Boss → 独特掉落表 + 解锁下个难度）        │
│                    ↓                                    │
│             转生 (Prestige)                               │
│ （重置进度 → 获得灵魂货币 → 永久加成 → 更强地重新开始）      │
└─────────────────────────────────────────────────────────┘
```

## 5. 战斗系统

### 自动战斗机制
- **驱动**: 1 秒间隔 Timer（1Hz Tick）
- **每 Tick 流程**:
  1. 存在存活敌人 → 玩家造成 DPS × 随机波动伤害
  2. 敌人反击 → 造成伤害给玩家
  3. 敌人死亡判定 → 掉落判断 → 生成物品 → 推进下一只
  4. 玩家死亡判定 → 复活机制
  5. 所有怪物清空 → 阶段通关 → Boss 战
  6. Boss 死亡 → 阶段清空 → 自动推进下一阶段

### 伤害公式（简化版）
```
物理伤害 = (武器最小..最大) × (1 + 力量/100 + 技能加成)
元素伤害 = 火伤 + 冰伤 + 电伤 + 毒伤
总伤害 = 物理伤害 + 元素伤害
暴击伤害 = 总伤害 × 暴击倍率 (暴击率触发)
DPS = 暴击伤害 × 攻速
火伤 = （面板伤害 - 敌人元素抗性）*持续时间
冰伤 = 面板伤害 - 敌人元素抗性（每次造成冰属性伤害都减少对方攻速）
电伤 = 面板伤害 - 敌人元素抗性（低频高伤，出伤有小概率打断对方行动条）
毒伤 = （面板伤害 - 敌人元素抗性）*中毒层数
```
